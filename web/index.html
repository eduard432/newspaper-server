News Paper API

API para obtener las portadas de los periódicos más importantes en México

Rutas disponibles:
  • GET /api/scrape?newsPaper={NEWSPAPER}&date={YYYY-MM-DD}
  • GET /api/newspapers
  • GET /api/cover?newsPaper={NEWSPAPER}&date={YYYY-MM-DD}
  • GET /api/covers/{YYYY-MM-DD || 'latest'}
  • GET /api/images/{FILENAME}

  • ({YYYY-MM-DD} => {YEAR-MONTH-DAY})
