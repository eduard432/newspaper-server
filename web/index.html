News Paper API

API para obtener las portadas de los periódicos más importantes en México

Rutas disponibles:
  • GET /api/newspapers
  • GET /api/cover?newsPaper={NEWSPAPER}&date={XXXX-XX-XX}
  • GET /api/covers/{XXXX-XX-XX || 'latest'}
  • GET /api/images/{FILENAME}

  • ({XXXX-XX-XX} => {YEAR-MONTH-DAY})
